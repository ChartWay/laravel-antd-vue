# Laravel 集成Ant Design Pro 组件

## 开箱即用
Ant Design Pro 是一个企业级中后台前端/设计解决方案，将其集成到Laravel 5.8项目中，配合Laravel框架使用

### Step1
```dotenv
npm install
```

### Step2
```dotenv
npm run dev 
```


### Step3
open url
```dotenv
http://doamin/admin
```

## 配置

`.env`文件中`MIX_ADMIN_BASE` 表示后台的地址前缀，默认是admin
